# 2025/12/15 首頁新聞區開發紀錄與總結

## 1. 目標與需求
- 首頁新聞區（國際、台灣、商業）自動化，風格對齊 Google News，主副新聞分明。
- 圖片、標題、連結、時間自動化，並支援 fallback。
- RWD、視覺一致性、維護性佳。

## 2. 主要開發步驟
- 以 Bootstrap 5 建立三大新聞卡片區塊，主副新聞結構清楚。
- 使用 Google News RSS + rss2json API，JS 動態 fetch 各區新聞，主副新聞自動填入。
- 圖片自動抓取 enclosure、thumbnail 或 description 內的圖片連結，若無則顯示預設圖。
- fetch 失敗或無資料時，主新聞顯示「暫無新聞」與「重新整理」按鈕，副新聞清空，區塊顯示提示。
- 圖片加上 onerror fallback，避免破圖。
- 副新聞 li 加上 border-bottom，提升可讀性。
- 時間欄位預留，支援 toLocaleString 格式，並討論「幾分鐘前」等相對時間顯示方式。
- CSS 統一抽離至 home_news.css，JS 抽離至 home_news.js，提升維護性。
- 測試本地圖片與網路圖片的破圖情境，確認 onerror 實用性。
- 討論 banner 圖片是否需 onerror，給予最佳實務建議。

## 3. 優化與細節討論
- 圖片縮放與解析度問題、object-fit 建議。
- RWD 微調、手機版圖片高度建議。
- 主副新聞分隔線、hover 效果、loading 狀態等細節建議。
- 多語系、主題色、API 流量控管等未來擴充性建議。
