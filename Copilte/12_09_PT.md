# Copilot 今日協作紀錄（首頁設計、市場指數、觀點區）

## 專案架構與設計
- 首頁導覽列（header）用 Bootstrap 5，RWD、登入/註冊按鈕、搜尋欄
- Banner 區塊用 Bootstrap Carousel，三張輪播（網站介紹、新聞、活動），文字置中、半透明黑底
- 市場指數區塊用表格展示重點指數（S&P500、NASDAQ等），加「查看更多」按鈕跳 market.html
- 表格資料用 JS 動態產生，漲跌幅自動判斷箭頭與顏色（▲紅、▼綠）
- 表格外層加框（border、rounded、shadow-sm、p-3），RWD最佳化（max-width、table-responsive）
- 按鈕用 w-100 w-md-auto，手機全寬、桌機自動縮回
- 討論純 HTML 與 JS 動態產生的差異，未來串 API 建議用 JS
- 其他區塊（新聞、觀點）用 Bootstrap row/col 排版

## 新增：觀點區
- 觀點區以卡片（card）方式呈現，標題「觀點摘要」
- 內容包含：作者/專欄名稱、來源、日期、主題、摘要
- 明確標註來源與外部連結（target="_blank" rel="noopener"），符合著作權規範
- 使用 Bootstrap Icons 作為視覺輔助
- 卡片設計有陰影、圓角、padding，RWD 友善

## 技術細節
- Bootstrap class 說明（display-4、fw-bold、mb-3、lead、mb-4等）
- aria-live="polite" 提升無障礙
- 三元運算子 ? : 用法
- table-hover 行 hover 效果可自訂
- object-fit:cover、object-position 控制圖片顯示
- RWD最佳化建議：table-responsive、max-width、字體大小、按鈕寬度
- 觀點區引用新聞摘要時，摘要內容非全文，並標註來源，外部連結加 rel="noopener"

## 互動流程
1. 討論首頁設計、導覽列、Banner輪播
2. 市場指數表格用 JS 動態產生，漲跌幅箭頭與顏色
3. 表格外觀美化、RWD優化
4. 討論純 HTML 與 JS 的差異及未來串 API 的便利性
5. 其他區塊排版、細節微調
6. 新增觀點區，重視著作權標註與引用規範
